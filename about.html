About This Project

About Me

My name is Justin Kramer, and I am currently a student in IT 3203: Intro to Web Development. This project was created as part of a class assignment to demonstrate my understanding of HTML, CSS, and JavaScript.

Project Overview

This website focuses on the evolution of HTTP over time. It explains how HTTP started as a simple protocol for transferring text documents and grew into a system capable of supporting modern web applications, multimedia content, and secure communication.

The site includes information about early versions of HTTP, modern improvements such as HTTP/2 and HTTP/3, and key concepts that explain how HTTP works behind the scenes.

What I Learned

While working on this project, I learned how to structure a multiple page website using HTML and style it using CSS. I also learned how to use JavaScript to create interactive features with the quiz included on my site.

Also, I gained a better understanding of how HTTP works and how it has evolved to meet the needs of the modern web. This project helped me learn more technical concepts while applying them to creating my own website.
